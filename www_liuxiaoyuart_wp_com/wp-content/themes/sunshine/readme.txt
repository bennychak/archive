主题简介
Sunshine采用黄色为主色调，两栏结构，适用于wordpress2.8主题，并兼容IE6、IE7、FIREFOX等主流浏览器。
在载入过程方面，优先载入文章区域，提升用户体验。
在SEO方面，对文章摘要做了优化处理，便于搜索引擎的收录。

下载&预览
http://wordpress.org/extend/themes/sunshine
Sunshine1.4版本，请点击此处下载（WP官网更新需要一段时间）

安装方法
1. 上传主题文件夹到wp-content/themes/目录
2. 进入管理界面点击外观-主题
3. 选择Sunshine模板

Description
This theme was designed and built by cuikai.
Soft and clean WP theme with 2 columns,Tested in Firefox, IE6 and IE7. Compatible up to Wordpress2.8,and SEO friendly.

Download&Preview
http://wordpress.org/extend/themes/sunshine

Install
1. Upload the theme folder via FTP to your wp-content/themes/ directory.
2. Go to Admin WP and select Design.
3. Select Sunshine theme.
